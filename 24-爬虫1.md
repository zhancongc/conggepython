## 请求方式

## requests库

```python
>>> import requests
>>> url = 'www.baidu.com'
>>> r = requests.get(url)
# 获取状态码，200表示一切正常
>>> r.status_code
200
>>> r.json
<bound method Response.json of <Response [200]>>
# 指定网页编码为utf-8
>>> r.encoding = 'utf-8'
# 获取网页内容，二进制流
>>> r.content
# 获取网页内容，文本信息，根据指定编码格式显示
>>> r.text
```
